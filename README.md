# How to compile
- make update to download the minilibx
- make
- make bonus

# How to launch the game
- ./cub3d maps/...
- ./cub3d_bonus maps/...

# How to play
- WASD to move around
- left and right arrow to change the camera direction
- M to display the minimap (bonus)
- O to open a door (bonus)
- T to show wall textures
